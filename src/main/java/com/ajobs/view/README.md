# Package

com.ajobs.view

## AjobsMenu.java
Provides a responsive menu that has controls for
primary navigation between views.

## AjobsViewType
Has four types of enum.
1. SpreadsheetView
2. LeaderboardView
3. StatsView
4. WipView

## LoginView.java
Provides Login Form for Signing in.

## MainView.java
Main UI container.
Container has AJobsMenu and views.
